# IBM Storage Ceph as a Service CSI driver overview 
{: #csi-driver-overview}

By using {{site.data.keyword.cephaas_full_notm}} CSI driver, Kubernetes PVs can be dynamically and statically provisioned for block storage to be used with stateful containers. 

Before performing operations, complete the [CSI driver set up](/docs/cephaas?topic=cephaas-csi-setup) steps. 

With {{site.data.keyword.cephaas_full_notm}} CSI driver, you can perform the following operations:

* [PersistentVolumeClaim (PVC) operations](/docs/cephaas?topic=cephaas-pvc-ops)
* [Snapshot operations](/docs/cephaas?topic=cephaas-csi-snapshot-ops)
* [Static provisioning](/docs/cephaas?topic=cephaas-static-provision)
